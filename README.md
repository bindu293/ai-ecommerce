# AI E-commerce Monorepo
i made this project
This is a full-featured, responsive, AI-enhanced e-commerce app.

- Frontend: React + Vite + Material-UI
- Backend: Node.js + Express + Firestore (Firebase Admin)
- Auth: Firebase Authentication (frontend) with backend token verification
- AI: OpenAI GPT for recommendations and product descriptions; DALLÂ·E for images (optional)
- Emails: Nodemailer (SMTP)

Quick start
- Copy env samples: backend/.env.example -> backend/.env; frontend/.env.example -> frontend/.env
- Install: npm install (from repo root)
- Seed sample data: npm run seed
- Run dev (both apps): npm start
